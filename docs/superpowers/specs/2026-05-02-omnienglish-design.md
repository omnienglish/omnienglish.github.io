# OmniEnglish CET-4 冲刺平台 — 设计规格

## 技术栈
- 单文件 HTML（index.html）
- 原生 CSS（OKLCH 色彩系统）
- 原生 JavaScript

## 页面结构

### 布局
- 桌面端：左侧 240px 固定侧边栏 + 右侧内容区
- 移动端：底部导航栏，内容全宽

### 冲刺首页（Dashboard）
1. **倒计时组件**：目标日期 2026.06.13，显示天/时/分/秒，进度条
2. **统计卡片**：已背单词、模考次数、连续天数
3. **四大功能入口卡片**：高频词汇墙、写作/翻译模考、真题长难句、CS 术语潜伏区
4. **今日任务清单**：可勾选，实时计数

### 功能子页面
- 高频词汇墙（即将上线）
- 写作/翻译模考（即将上线）
- 真题长难句（即将上线）
- CS 术语潜伏区（即将上线）

## 设计系统
- 浅色主题，OKLCH 色彩
- 主色：深蓝 oklch(0.52 0.19 250)
- 字体：JetBrains Mono（标题）+ DM Sans（正文）
- 12px 圆角卡片，8px 间距网格
